# React Practice Work

Collection of small React/Vite practice projects.

## Projects
- `2 create-react-componenet/` - Vite + React starter app with a simple counter to practice component structure, state, and events.
- `Projects/Calculator/calci-v1/` - Basic calculator UI with number/operator buttons, clear (C), and evaluate (=) handling, including error fallback.
- `Projects/Clock/Bharat-clock_v-1/` - Minimal "Bharat Clock" page with a static time display (placeholder for a live clock).
- `Projects/FoodApp/FoodApp/` - "Healthy Food" list app: add items with Enter, show empty-state message, and mark items as bought.
- `Projects/Todo-App/1-Todo-app-Version-1/` - Todo list with task name + due date, add/delete actions, and localStorage persistence.

## Run a project
Each folder is a standalone Vite app. From a project folder:

```bash
npm install
npm run dev
```

See each project folder's README for details.

## Notes
Some folders include `node_modules` already.
